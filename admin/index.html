¿Cómo publicar reseñas aprobadas?

  1. 1
    Apruebe reseñas en la pestaña "Pendientes". Las aprobadas se acumulan en memoria de esta sesión.
  2. 2
    Descargue reviews.json con el botón de abajo. El archivo incluye las reseñas actuales del sitio más las que acaba de aprobar.
  3. 3
    Reemplace el archivo en su proyecto:
    docDianaSanchez/reviews/reviews.json
  4. 4
    Haga commit y push en su terminal: 
    cd ~/docDianaSanchez
git add reviews/reviews.json
git commit -m "chore: update approved reviews"
git push
